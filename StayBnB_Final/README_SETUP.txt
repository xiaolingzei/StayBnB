
StayBnB_Final - Setup Guide
1. Extract this folder into your XAMPP htdocs (e.g. D:\xampp\htdocs\StayBnB_Final)
2. Start Apache and MySQL in XAMPP.
3. Open phpMyAdmin (http://localhost/phpmyadmin) and import 'staybnb_db.sql' from project root.
4. Ensure config/db_connect.php has correct DB credentials (default is root with no password).
5. Visit http://localhost/StayBnB_Final/
